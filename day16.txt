1. NUMPY(넘파이)
(1) 개요
    대부분의 데이터는 행과 열로 이루어져 있음
    행과 열의 데이터를 빠르게 수정
(2) 특징
    컴퓨터 과학을 위한 기본적인 파이썬 패키지
    n차원의 데이터를 빠르게 처리
    C, C++, 포트란으로 만들어짐
    선형대수학을 빠르게 계산
    강력한 브로드캐스팅 기능
(3) 기본 개념
    스칼라 : 값이 하나만 있는 데이터
    벡터 : 하나의 행에 여러 개의 데이터
    매트릭스 : 행과 열이 있는 데이터
(4) 참고 사이트
    http://www.numpy.org
    http://aikorea.org/cs231n/python-numpy-tutorial

2. NUMPY 사용법
기본적으로 import numpy as np
(1) 행렬의 생성 1
    리터럴(문자)로 행렬을 생성
    리터럴 : 변수 선언 시 직접 값을 입력하여 생성하는 방법
    numpy에서는 list보다 속도가 빠른 ndarray 데이터 타입을 사용

(2) 행렬 데이터의 선택
    행렬 데이터를 수정하거나, 가져오거나, 자를 때

(3) 행렬의 생성 2
    zeros : 행렬을 만들고 0으로 채움, dtype으로 데이터 타입 설정 가능
    ones : zeros의 1 버전

(4) arange 만들기
    range와 사용 방법이 동일하나 ndarray 타입으로 만들어짐

(5) linspace, logspace
    linspace : 설정한 범위에서 선형적으로 분할한 위치에 대한 값
    logspace : 설정한 범위에서 로그함수적으로 분할한 위치에 대한 값

(6) Numpy Random
    seed : 랜덤변수의 설정값
    rand : 균등분포(Uniform)로 난수
    randn : 정규분포(Gaussian)로 난수
    randint : 정수로 난수를 발생(uniform)
    shuffle : 행렬 데이터를 섞어줌
              가장 높은 차원
    choice : 특정 범위의 숫자를 선택(확률 설정이 가능)
             확률의 모든 합은 1이 되어야 함
