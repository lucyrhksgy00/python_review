1. 문자열(String)
   '' "" 모두 사용 가능 cf) C, Java : 'a', "abcde"
   str.format() : "내 이름은 {}이다".format('이름없음')
   (1) find(문자열) : 문자열을 찾고 어디에 위치하는지 인덱스 리턴
       ex) "가나다라마바사".find('나다') = 1
   (2) lstrip, rstrip, strip : 왼쪽, 오른쪽, 양쪽으로부터 해당 문자열 삭제
       다른 언어에서는 일반적으로 ltrim, rtrim, trim
   (3) title() : 각 단어의 첫 글자를 대문자로
   (4) upper(), lower() : 대문자로, 소문자로

2. Class와 객체지향
(1) 클래스와 객체, 객체지향
    변수 : 하나의 값만 저장
    배열 : 여러 개의 값을 저장
    구조체(struct) : 여러 형태의 값을 저장
    클래스 : 여러 형태의 값 + 기능(function)

(2) 정의
    클래스(Class) : 설계도, 상상 속에서만 존재하는 것
    객체(Object) : 설계도로 만든 실제 작품
    cf) Apple, An apple

(3) 용어
    1) 클래스
       값(변수)와 기능(함수)로 구성
    2) 메서드(Method)
       클래스가 가지고 있는 함수는 함수라고 하지 않고 메서드라고 함
       메서드의 초기 이름은 멤버 함수
    3) 변수(Field)
       클래스가 가지고 있는 변수는 필드라고도 함
    4) 객체(Object)
       클래스(설계도)로 만들어지는 것
       클래스를 정의하고 클래스()를 하면 객체가 생성됨 cf) new 클래스()
    5) 인스턴스(Instance)
       한마디로 객체
       클래스와의 관계를 표현할 때 인스턴스라는 말을 씀
       ex) 자동차 클래스의 객체 트럭1, 택시1
           트럭1 = 자동차()
           택시1 = 자동차()
           트럭1은 객체이다.
           트럭1은 자동차의 인스턴스이다.

(4) 클래스 정의 문법
   class 클래스명:
       <statement1>
       .
       .
       .
       <statementN>

(5) 객체 생성 문법
    객체1 = 클래스명()

(6) 첫 번째 클래스와 인스턴스 만들어보기
    class House:

    myhouse = House()
