10. append : 데이터 합치기
    df1과 df2를 합치는 방법 : df1.append(df2)
    append 후에 reset_index()라는 것이 필요

11. concat : row 또는 column 기준으로 데이터 합치기
    df3 = pd.concat([df1, df2], axis=0 or 1)

12. groupby
    특정 컬럼의 중복되는 row를 합쳐서 새로운 데이터 프레임을 만드는 방법
    size() : 개수
    agg(*) : * - min, max, mean, median, sum

13. select
    데이터를 고르는 방법
    head() : 위쪽 다섯 개
    head(n) : 위쪽 n개
    tail() : 아래쪽 다섯 개
    tail(n) : 아래쪽 n개

    cf) UNIX, LINUX : tail -f 20220906.log

14. csv, excel
(1) xlrd, openpyxl, Xlsxwriter 설치
(2) excel write
    df.to_excel(파일명, sheet_name='Sheet1', index=False,
                engine='xlsxwriter', encoding='utf-8-sig')
